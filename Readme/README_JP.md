自分がSteamにいくら使ったか集計するのは非常に手間がかかります。公式の利用明細はページ分割表示されており、手動で「さらに読み込む」を連打し、購入と返金の注文を一つずつ仕分けて金額を手計算する必要があります。

この問題を解決するため、軽量な油猴スクリプト **Steam消費金額合計確認** を作成しました。全ての利用明細を自動取得し、注文種別を判別して総支出額と返金総額を自動計算します。複数地域の通貨、多言語表示、ローカルキャッシュ、複数のフリーズ防止機構に対応しています。

## スクリプトの主な機能
### ワンクリックで全明細集計✨
Steamの[購入履歴ページ](https://store.steampowered.com/account/history)を開く，にアクセスすると、ページ下部に「消費統計を確認」ボタンが自動表示されます。クリックすると完全な集計処理が自動実行されます。
- 「さらに読み込む」ボタンをループで自動クリックし全履歴明細を取得。無限読み込みによる画面フリーズを防ぐため最大120ページの制限を設置
- ページの通貨記号を自動判別し、全世界の各リージョンの通貨に対応
- 購入注文と返金注文をスマートに分類し、3種類の核心金額を自動算出
	- 総支払額：後に返金された注文を含む全ての購入金額合計
	- 累計返金額：全返金注文の金額合計
	- 実純支出額 = 総支払額 − 累計返金額（実際に支払った金額）

### ローカルキャッシュ高速化機能⚡
- 集計結果を5分間ローカルにキャッシュ。短時間内に連続クリックしても再度ページを読み込まず、動作速度を大幅に向上させ画面のカクつきを抑制
- キャッシュ期限切れ後に再度ボタンをクリックすると、全明細を再取得し最新データを再計算

### 多言語自動切り替え対応🌍
Steamページの表示言語を自動読み取り、ポップアップやボタンのテキストが自動切り替わります。現在3言語に対応：
- 簡体中国語（zh-CN）
- 繁体中国語（zh-TW）
- 英語

### 複層フリーズ防止・エラー回避設計🛡️
- 各ページ読み込み間隔を450msに設定し、Steam側のデータ描画時間を確保、リクエスト過多による制限を回避
- MutationObserverによるノード監視 + 800ms周期のDOM定期スキャンの二重構造でボタンを正常表示
- 30秒後に定期DOMスキャンを自動停止し、ブラウザのリソース消費を削減
- ページ読み込み異常をグローバルに捕捉、エラーポップアップを表示しページ更新による再試行を誘導
- ローディングロックにより重複クリックを遮断、明細の重複リクエストを防止

## カスタム設定パラメータ
コード上部の`cfg`オブジェクトを編集することで動作パラメータを調整可能です。

```javascript
const cfg = {
    maxPage: 120,    // 明細読み込み最大ページ数、明細データが多い場合は数値を上げる、上限は200
    wait: 450,       // 1ページ読み込み待機時間、ネットワークが不安定な場合は800～1000に調整
    cacheTime: 300000, // キャッシュデータ有効期間(ミリ秒)、300000は5分間に相当
    domTick: 800     // ページ要素走査間隔、通常変更する必要なし
};
```

# インストールと使用手順
## 事前準備
ブラウザに **Tampermonkey（油猴）** 拡張機能をインストールしてください。Chrome、Edge、Firefox、Operaに対応しています。

## ワンクリックでスクリプトをインストール
Greasy ForkスクリプトURL：<https://greasyfork.org/scripts/583431>
リンクを開き「このスクリプトをインストール」をクリックするだけで油猴に導入できます。

## 使用手順
- Steamウェブサイトにログイン
- 利用履歴ページを開く：https://store.steampowered.com/account/history
- ページ下部に「消費統計を確認」ボタンが表示されます
- ボタンをクリックし、全明細の自動読み込み完了を待機
- 読み込み完了後、総支払額・返金総額・実純支出額がポップアップに表示されます

# 最後に
本スクリプトは完全にローカルで動作し、明細やアカウント情報を第三者サーバーにアップロードすることは一切ありません。広告やバックグラウンド通信もなく、軽量でページに干渉しません。

長年のSteamゲーム支出をすぐ集計したい方はぜひインストールしてお試しください。使用上のトラブルがあればコメント欄にご記入ください。
