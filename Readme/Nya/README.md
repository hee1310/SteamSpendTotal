### 一键召出全账单演算法阵✨
开启Steam [交易记录](https://store.steampowered.com/account/history) 
页面会自动生成「查看消费统计」猫爪按钮，轻点之后一切交给本喵处理～
- 自动循环激活「加载更多」符文，扒取完整过往交易卷宗，最多遍历120层卷宗位面，安分守己不乱闯服务器
- 自主识别全世界各类货币刻印，任何地区的Steam账号都能使用喵秘术
- 聪慧分辨氪金、退款两类卷宗，算出三份核心金币数据：
	- 总献祭金额：所有下单原始消耗金币（包含后续返还的款项）
	- 总返还金额：所有退回钱包里的金币数额
	- 次元净消耗 = 总献祭金额 − 总返还金额（主人真正献给Steam的小钱钱）

可自行修改脚本开头 `cfg` 配置项，调整法阵运转规则～

```javascript
const cfg = {
    maxPage: 120,    // 最大加载账单页数，账单极多可上调，建议不超200
    wait: 450,       // 每页加载等待毫秒，网络差可增大至800~1000
    cacheTime: 300000, // 统计缓存有效期ms，300000=5分钟

};
```
