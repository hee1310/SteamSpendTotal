Steamにどれだけお小遣いを散財しちゃったか計算しようとするたび、めちゃくちゃ手間がかかるの～
公式の利用明細は手動で「さらに読み込む」を連打しなきゃいけない上、購入分と返金分を自分で分けなきゃならなくて、ネコ脳がボーッとしちゃう(´• ω •`)ﾉ
だからこのネコ娘が、超軽量もふもふTampermonkeyスクリプト **Steamお買い物金額集計アシスタント** を自作したの！
全履歴明細を自動取得、注文種類を自動仕分け、合計消費金額と返金額を自動計算。複数通貨・多言語対応、ローカルキャッシュで高速動作、ページが固まることなく静かに動いてくれる！

## ネコ娘限定もふもふ機能
### ワンクリック全明細スキャン集計✨
Steamの[購入履歴ページ](https://store.steampowered.com/account/history)
ページに「消費統計を確認」のもふもふボタンが自動出現するから、クリックしたら全部このネコ娘にお任せ！
- 「さらに読み込む」を自動連打して全履歴を取得、上限120ページまで大人しく動いて暴走しない
- 世界各国の通貨記号を頭良く判別、全世界のSteamで完璧に動作
- ネコ娘仕様の頭脳で購入・返金注文を分類、大事な3つの金額を一瞬で算出：
	- 総合消費額：全注文時の元々の支払金額（後で返金申請した注文も含む）
	- 総合返金額：ウォレットに戻ってきた全金額の合計
	- 実際の純消費金額 ＝ 総合消費額 − 総合返金額（本当にお財布から消えたお金）

### ローカルキャッシュでサクサク高速動作⚡
- 集計データを5分間キャッシュ保存、短時間に再確認する時は再度ページを読み込まず滑らかに表示
- キャッシュ期限が切れた後ボタンを押すと、最新の利用明細を再取得して再計算してくれる

### 多言語自動切り替え機能🌍
Steamサイトの表示言語に合わせて自動変更、ボタンもポップアップも全部もふもふ猫娘仕様：
- 簡体中国語
- 繁体中国語
- English

### 多重フリーズ防止シールド🛡️
- 1ページ読み込むごとに450ms間隔を空けて優しく取得、Steamサーバーに負荷をかけない
- 二重監視システムで統計ボタンが必ず表示され、800msごとに定時確認で超安定
- 30秒間操作がなければ自動で監視を停止、ブラウザのリソースを消費しない
- データ取得時にエラーが発生したら、「ページを更新してから再試してみて」と柔らかくお知らせ
- 二重クリック遮断機能搭載、重複したデータリクエストを送信しない

## カスタムパラメータ設定（猫娘専用モード）
コード先頭の `cfg` 部分を編集するだけで、スクリプトの動作設定を自由に調整可能～

```javascript
const cfg = {
    maxPage: 120,    // ニャンこが取得できる請求書最大ページ、多いなら上げて上限200
    wait: 450,       // 1ページ読み込み待機時間、回線重い時800～1000に変更
    cacheTime: 300000, // 統計キャッシュ有効時間(ms)、300000は5分間
    domTick: 800     // DOM監視周期、普段触らなくて大丈夫ニャ
};
```

# インストール＆使い方もふもふ講座
## 事前に必要なもの
まずブラウザに **Tampermonkey（油猴）** 拡張機能をインストール！Chrome、Edge、Firefox、Opera全て対応！

## ワンクリック猫娘スクリプト導入
Greasy Fork 専用スクリプトURL：https://greasyfork.org/scripts/583431
リンクを開いて「このスクリプトをインストール」をクリックするだけ、一発でTampermonkeyに導入完了！

## 超簡単操作手順
- Steamウェブ版アカウントにログイン
- 利用履歴専用ページを開く：https://store.steampowered.com/account/history
- ページ下部に「消費統計を確認」のもふもふボタンが自動出現
- そっとクリックして、このネコ娘が全履歴明細を読み込むのを待つ
- データ取得完了後、総合消費額・総合返金額・実際の純消費金額がポップアップ表示される
