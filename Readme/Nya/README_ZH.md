## 腳本超萌功能
### 一鍵載入全部帳單統計✨
打開Steam [消費紀錄頁](https://store.steampowered.com/account/history)
頁面會自動冒出「查看花費統計」按鈕，點下去就交給本喵～
- 自動點「載入更多」抓完整歷史帳單，最多120頁乖乖不亂跑
- 自動認識各國貨幣符號，全世界Steam都能用
- 聰明分消費&退款訂單，算出三個重要金額：
	- 總支出：所有下單的原始金額（含後來退款的）
	- 總退款：所有退錢回來的金額
	- 實際淨花費 = 總支出 − 總退款（真正花掉的錢錢）

### 本地快取超快速⚡
- 統計結果快取5分鐘，短時間重點不用重新載入，順順不卡頓
- 快取過期後再點，會重新抓最新帳單重新計算

### 多語言自動切換🌍
跟著Steam頁面語言自動換，按鈕&彈窗都萌萌噠：
- 簡體中文
- 繁體中文
- English

### 多重防卡死保護機制🛡️
- 每頁間隔450ms慢慢載，不搶Steam資源
- 雙重監聽保證按鈕一定出現，800ms定時檢查超穩定
- 30秒自動關閉檢查，不佔用瀏覽器效能
- 遇到異常會溫柔提醒「刷新頁面重試」
- 防止重複點擊，不重覆請求資料

## 自訂參數設定（貓娘模式）
可自行修改程式頂端 `cfg` 調整運行參數～

```javascript
const cfg = {
    maxPage: 120,    // 本喵讀取帳單最大頁數，帳單過多可調高，上限200喵
    wait: 450,       // 單頁載入等候時長，網路遲鈍調至800~1000喔
    cacheTime: 300000, // 快取資料保鮮時限(毫秒)，300000等同五分鐘
    domTick: 800     // 頁面元件巡檢間隔，平常無需調整喵
};
```

# 安裝與使用小教程
## 前置條件
瀏覽器先裝 **Tampermonkey（油猴）** 擴充功能，Chrome、Edge、Firefox、Opera都支援喔！

## 一鍵安裝腳本
Greasy Fork 腳本網址：https://greasyfork.org/scripts/583431
打開連結點「安裝此腳本」，一鍵匯入油猴～

## 使用步驟（超簡單）
- 登入Steam網頁版
- 打開消費紀錄頁：https://store.steampowered.com/account/history
- 頁面下方出現「查看花費統計」按鈕
- 點下去等待本喵幫你載入全部帳單
- 載完彈窗顯示總支出、退款、實際淨花費
